# Уровни аналитиков

**Источник:** [avito-tech/playbook/analytics-levels.md](https://github.com/avito-tech/playbook/blob/master/analytics-levels.md)

## Грейды

| Уровень | Название | Фокус |
|---------|----------|-------|
| A1 | Младший аналитик | Сбор данных, простые отчёты |
| A2 | Аналитик | Продуктовая аналитика, когорты |
| A3 | Старший аналитик | Метрики кластера, дашборды |
| A4 | Head of Analytics | Аналитическая стратегия |

## Роль в качестве

- Метрики качества: баги на MAU, конверсия ошибок, retention после релиза
- Дашборды для мониторинга: Apdex, availability, response time
- A/B тесты: проверка гипотез с учётом качества
- Data-driven приоритизация технического долга

## Применение к кейсу

- Аналитик — участник ATDD (Три амиго)
- Помогает с метриками для оценки качества
- Строит дашборды: баги по командам, MTTR, инциденты
