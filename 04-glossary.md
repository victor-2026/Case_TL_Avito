# Glossary (Все акронимы — кратко)

## Monetization & Product

| Term | Meaning |
|------|---------|
| **MNZ** | Кластер монетизации |
| **PFP** | Pay-For-Performance — оплата за результат |
| **CPL** | Cost Per Lead — стоимость за лида |
| **CPA** | Cost Per Action — стоимость за действие |
| **ARPU** | Average Revenue Per User — выручка на пользователя |
| **NPS** | Net Promoter Score — индекс лояльности |
| **TTV** | Time-to-Value — время до получения ценности |

## Engineering & Process

| Term | Meaning |
|------|---------|
| **TDR** | Tech Design Review — дизайн-ревью архитектуры |
| **ZBP** | Zero Bug Policy — не копим баги |
| **SPT** | Service Performance Testing — тестирование производительности |
| **ATDD** | Acceptance Test-Driven Development — приёмочные тесты до кода |
| **TMM** | Team Maturity Model — модель зрелости команды |
| **RBT** | Risk-Based Testing — тестирование на основе рисков |
| **NFR** | Non-Functional Requirements — нефункциональные требования |
| **RCA** | Root Cause Analysis — анализ корневых причин |
| **DXF** | Developer Experience Framework — фреймворк для DX |

## Metrics & Quality

| Term | Meaning |
|------|---------|
| **DF** | Deployment Frequency — частота деплоя (DORA) |
| **LT** | Lead Time for Changes — время от коммита до продуктива (DORA) |
| **MTTR** | Mean Time to Restore — среднее время восстановления (DORA) |
| **CFR** | Change Failure Rate — доля неудачных релизов (DORA) |
| **QS** | Quality Score — агрегированная оценка качества |
| **SLA** | Service Level Agreement — соглашение об уровне сервиса |
| **SLI** | Service Level Indicator — индикатор уровня сервиса |
| **SLO** | Service Level Objective — целевой уровень сервиса |
| **Apdex** | Application Performance Index — индекс производительности приложений |

## Technology

| Term | Meaning |
|------|---------|
| **RPS** | Requests Per Second — запросов в секунду |
| **AUC** | Area Under Curve — метрика качества ML-модели |
| **PBT** | Property-Based Testing — property-based тестирование |
